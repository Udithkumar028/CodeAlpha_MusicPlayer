# Music Player

A simple music player project developed using HTML, CSS and JavaScript.

## Features

* Play and pause functionality
* Next and previous song controls
* Playlist support
* Autoplay next song
* Progress bar
* Volume control
* Keyboard shortcuts
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Purpose

This project was created to practice audio handling, DOM manipulation and frontend UI development using JavaScript.
